# LearnAuth
AI that Detects if a YouTube Video Actually Teaches

```Developed at ACE VIPS under Research and Development Domain 2025```

**LearnAuth** is an AI-powered project that analyzes YouTube videos to determine whether the speaker is genuinely teaching or just talking.  
It uses **Google Gemini API** for natural language understanding and integrates seamlessly through both a **web app** and a **Chrome extension**.
